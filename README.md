# NutriHealth web service

<h3>Login</h3>

```
content-type  : json
POST          : api.nutripal.site/login
body          : {email, password}
```

<h3>Register</h3>

```
Content-type   : json
POST           : api.nutripal.site/register
body           : {email,nama_depan,nama_belakang,password,tanggal_lahir}
tanggal lahir date "tahun-bulan-tgl"
ex : "2001-03-31"
```
